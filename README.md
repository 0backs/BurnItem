## 설치 방법
BurnItem.jar 플러그인을 plugins 폴더 안에 배포해 주세요.

## 버전 안내
1.21.4 버전을 기준으로 제작되었으며, 특별한 능력이 없기에 상위 버전은 호환될 수도 있습니다.

## 권한
##### burnitem.admin = 용광로 적용 시 소모되는 아이템을 설정하는 권한
